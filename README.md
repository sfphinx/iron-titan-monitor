# iron titan monitor

This is a simple script that is design to monitor a villages.dat file to let you know if you are building your iron titan properly.

It monitors the villages.dat file and if you present it with the starting x, y, and z coordinates.

## Requirements

Node.JS or Docker

## Running 

### Node.JS

```bash
node monitor.js villages.dat X Z Z
```

#### Example

```bash
node monitor.js villages.dat 210 87 211
```

### Docker

```bash
docker pull sfphinx/iron-titan-monitor

docker run -it --name="iron-titan-monitor" \
  -v /data:/folder/on/host/where/villages.dat \
  sfphinx/iron-titan-monitor /data/villages.dat 210 87 211
```
